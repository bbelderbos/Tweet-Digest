README TWEET DIGEST
====================

requirements: Twitter 1.1 API

create a config.php file with your keys/ secrets

$ cat config.php
$settings = array(
    'oauth_access_token' => "aaa",
    'oauth_access_token_secret' => "bbb",
    'consumer_key' => "ccc", 
    'consumer_secret' => "ddd",
);
