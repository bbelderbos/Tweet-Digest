README TWEET DIGEST
====================

Requirements: Twitter 1.1 API (create a twitter app)

Create a file in the toplevel src directory called config.php with the obtained credentials:

    $ cat config.php
    $settings = array(
      'oauth_access_token' => "aaa",
      'oauth_access_token_secret' => "bbb",
      'consumer_key' => "ccc", 
      'consumer_secret' => "ddd",
    );
